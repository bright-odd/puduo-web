# IT Solutions Company Landing Page

A modern, responsive landing page for an IT Solutions company built with HTML, CSS, and JavaScript.

## Features

- **Modern Design**: Clean, professional layout with gradient accents and smooth animations
- **Fully Responsive**: Works seamlessly on desktop, tablet, and mobile devices
- **Interactive Elements**: Smooth scrolling, hover effects, and animated counters
- **Performance Optimized**: Lightweight and fast-loading
- **Accessible**: Semantic HTML and proper ARIA attributes

## Sections

1. **Navigation Bar**: Fixed header with smooth scroll navigation
2. **Hero Section**: Eye-catching introduction with animated elements and statistics
3. **Services Section**: Grid layout showcasing 6 key services
4. **About Section**: Features and benefits with visual elements
5. **Testimonials**: Client reviews and feedback
6. **Call-to-Action**: Prominent section to encourage user engagement
7. **Footer**: Comprehensive site links and contact information

## Getting Started

1. Open `index.html` in your web browser
2. No build process or dependencies required
3. All assets are self-contained

## Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #2563EB;
    --primary-dark: #1D4ED8;
    /* ... */
}
```

### Content
Edit the HTML content in `index.html` to match your business needs.

### Animations
Modify animation timings and effects in `styles.css` and `script.js`.
